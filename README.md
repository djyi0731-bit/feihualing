# 诗词飞花令 · 车载声控版（Demo + 工程骨架）

按你的选择交付：**Android 车机 + 第三方语音 SDK + 先搭可运行 Demo**。
由于当前环境无法编译 Android 工程，Demo 用「浏览器可运行、真实中文声控」的原型快速验证交互与规则；同时提供 Android 工程骨架作为最终装车代码基础。

## 如何运行 Web Demo（立即体验）
1. 用 **Chrome / Edge** 打开 `index.html`（手机或电脑均可，需联网，中文识别走云端）。
2. 允许「麦克风」权限 → 点「开始游戏」→ 系统给出令字与示范句。
3. 点 **🎤 听我说**，说出含令字的诗句（例：令字「月」，说「床前明月光，疑是地上霜」）。
4. 系统判定：确为真实诗词/元曲、含令字且未重复 → ✓ 加分并展示该诗全文；否则 ✗ 提示（如「这不是已知的诗词或元曲哦」）。
5. 不支持语音的浏览器会自动降级为「手动输入框」，照样能玩。

## 当前占位规则（待你替换）
- **单字令**：每局随机一个令字，双方所有诗句须包含该字。
- **必须为真实诗词/元曲**：所说内容须能对应题库中的真实诗句（不再是「含字即算对」），否则判错并提示「这不是已知的诗词或元曲哦」。
- **不重复**：用过的句子不可再用。
- **计分**：每接对一句 +10。
- **题库**：内置 10 个令字 × 数句诗，仅作演示，非完整唐诗库。

## 技术路线小结
- 声控：语音转文字（ASR）→ 文本匹配判定，车载建议第三方 SDK（离线/云端混合）。
- 判定：本地题库 + 规则引擎，无需实时联网（语音识别按需联网）。
- 车载：横屏大屏、高对比、以听/说为主、尽量少看屏幕，符合行车安全。
